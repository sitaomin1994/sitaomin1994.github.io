Projects

Post-HCT Survival Predictions using Machine Learning and Deep Learning | 2025

Kaggle CMBTR Challenge, Silver Medal (74/3084)

  • The aim of this project is to develop advanced survival predictive models for almost 30,000 allogeneic hematopoietic cell transplant (HCT) patients collected from CIBMTR.
  • Proposed a solution for Silver Medal by ensembling multiple ML/DL models including Light GBM with Kaplan Meier transformation and XGBoost-based cox model, accelarated failure model and deep learning based survival prediction model. | Solution Report: CMBTR-Survivor

Predict Protein-molecule Interaction for New Medicines using Deep Learning | 2024

Kaggle BELKA Challenge, Bronze Medal (178/1950)

  • The aim of this project is to develop predictive models for estimating small molecule binding affinity to protein targets using the BELKA dataset to accelerate drug discovery.
  • Developed an ensemble approach combining 1D CNN, Transformer, and Mamba architectures with character-level tokenization of SMILES strings for molecular sequence encoding to achieve robust multi-label classification performance. | Solution Report: BELKA-2024

Financial Crime Prevention using Secure AI | 2024

US-UK PETs Challenge, 1st Place (10,000 USD)

  • The aim of this project is to develop privacy-preserving federated learning solutions for detecting anomalous financial transactions to combat international money laundering.
  • Designed privacy-preserving federated XGBoost using Bloomfilter for secure anomalous financial transactions detection. | Solution Report: Scarlet-PETs
